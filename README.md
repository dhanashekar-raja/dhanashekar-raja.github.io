# Portfolio Starter

This is a static portfolio website for Dhanashekar Raja, a Senior Technical Writer with a product-facing career direction.

## Files

- `index.html`: page structure and starter content
- `styles.css`: layout, color system, responsive design, and motion
- `script.js`: mobile menu and scroll reveal

## Next customization steps

1. Review and refine the wording in `index.html` so it matches your preferred tone.
2. Add direct links to writing samples, documentation screenshots, or case studies if you want stronger proof of work.
3. Open `index.html` in a browser to review the site locally.
4. Publish the site with GitHub Pages or Netlify when you are ready.
5. If using GitHub Pages, keep `index.html` at the repository root so it can be served directly as a static site.
